# Lesson 02 — Variables, Methods, and Bools

Variables, methods, and bools are the basic building blocks of programming logic.

---

## Core parts
- Variables: store data (numbers, text, states)
- bool: represents a condition (true / false)
- Methods: define actions executed by the program

---

## Typical flow
1. Variables store the current state
2. Methods read or modify these variables
3. bool checks conditions
4. Logic executes based on these conditions

---

## Key idea
Programming is based on **state and actions**.  
Variables hold information, methods act on it, and bool controls decisions.
